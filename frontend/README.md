# ozma 前端

前端仓库由 [turborepo.dev](https://turborepo.dev/) 管理。

## 快速开始

环境要求：

* Node.js 20+ (推荐lts版本)
* pnpm 10.29.1
* turbo 2.7.6

### 1. 安装Node.js 20+

```bash
    fnm install lts

    fnm 
```

### 2. 检出 Git 仓库并切换到 `frontend` 目录

```bash
    # 克隆仓库
    git clone

    # 切换目录
    cd frontend
```

### 3. 安装全局依赖
```bash
    pnpm add turbo@2.7.6 --global
```
